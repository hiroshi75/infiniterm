# infiniterm 改善・新機能アイデア集

## 凡例

- 🔴 高優先度 — ユーザー体験に直結する改善
- 🟡 中優先度 — あると便利な機能強化
- 🟢 低優先度 — 面白い・差別化につながる機能

---

## 1. ターミナルコア機能の強化

### 🔴 ペイン分割 (Split Pane)

現在はタブのみ。tmux なしで画面分割できると大きな差別化になる。

- 水平/垂直分割
- ドラッグでリサイズ
- ペイン間フォーカス移動 (Ctrl+Alt+矢印)
- ペインごとに異なるシェル/SSH セッション

### 🔴 セッション復元

アプリ再起動時にすべてのタブ・作業ディレクトリが失われる。

- タブ一覧・CWD・シェル種別を永続化
- 起動時に前回のセッションを自動復元 (オプション)
- スクロールバック履歴の保存/復元

### 🟡 コマンドパレット

VS Code 風の `Ctrl+Shift+P` で全機能にアクセス。

- 検索可能なコマンド一覧
- 最近使ったコマンド履歴
- キーバインド表示
- SSH プロファイル直接接続

### 🟡 画像プロトコル対応

- Sixel グラフィックス
- iTerm2 インライン画像プロトコル
- `imgcat` 相当のコマンドで画像をターミナル内表示

### 🟡 OSC 8 ハイパーリンク

`ls --hyperlink=auto` 等で生成されるリッチリンクの表示・クリック対応。

### 🟢 リガチャフォント対応

Fira Code, JetBrains Mono 等のプログラミング用リガチャを表示。

---

## 2. UI/UX の改善

### 🔴 ステータスバー

ターミナル下部に情報バーを追加。

- 現在の CWD
- シェル名・PID
- Git ブランチ (CWD 連動)
- SSH 接続先ホスト名
- エンコーディング表示

### 🔴 通知システム

- バックグラウンドタブでの出力完了通知 (長時間ビルド等)
- ベル文字 (`\a`) でのデスクトップ通知
- タブアイコンでの未読/活動表示

### 🟡 ドラッグ & ドロップ

- ファイルをターミナルにドロップ → パスを挿入
- タブのドラッグ並べ替え
- タブを外にドラッグ → 新ウィンドウ化

### 🟡 スニペットマネージャー

よく使うコマンドを保存・呼び出し。

- カテゴリ分け (git, docker, ssh, etc.)
- 変数プレースホルダー (`${HOST}` 等)
- 検索・フィルタ
- インポート/エクスポート

### 🟡 マルチウィンドウ対応

現在はシングルウィンドウ。複数ウィンドウを開けるようにする。

### 🟢 カスタムテーマエディタ

GUI でテーマの色をカスタマイズ・保存。

- カラーピッカー UI
- プレビュー画面
- テーマのインポート/エクスポート (JSON)
- iTerm2/Windows Terminal テーマ形式のインポート

### 🟢 背景画像/透過設定の強化

- 背景画像の設定
- ぼかし (blur) 強度の調整
- タブごとに異なる背景

## 3. SSH の強化

### 🔴 SSH Agent 転送

`ssh-agent` 連携で鍵パスフレーズの再入力を不要に。

### 🟡 ProxyJump / 踏み台サーバー対応

多段 SSH 接続 (`-J` 相当) をプロファイル設定で指定。

### 🟡 ポートフォワーディング UI

- ローカル/リモートフォワーディングの GUI 設定
- SOCKS プロキシ作成
- アクティブなトンネル一覧表示

### 🟡 ホスト鍵検証 UI

初回接続時のフィンガープリント確認ダイアログ。既知ホストの管理画面。

### 🟢 SSH Config インポート

`~/.ssh/config` からプロファイルを自動インポート。

### 🟢 SFTP ファイルブラウザ

SSH 接続先のファイルを GUI で閲覧・転送。

---

## 4. Web ターミナルの強化

### 🔴 HTTPS (TLS) 対応

自己署名証明書 or Let's Encrypt で暗号化通信。特に公開ネットワークでの安全性向上。

### 🟡 ファイル転送

- ブラウザ → ターミナルへのファイルアップロード
- ターミナル → ブラウザへのファイルダウンロード
- ドラッグ & ドロップ対応

### 🟡 クリップボード双方向同期

デスクトップとモバイルブラウザ間でクリップボードを同期。

### 🟡 マルチタブ対応

Web ターミナルでもタブを切り替え可能に (現在は単一セッション)。

### 🟢 音声入力

モバイルブラウザの音声認識 API でコマンド入力。

### 🟢 PWA 対応

Web ターミナルを PWA としてインストール可能に。ホーム画面にアイコン追加。

---

## 5. ターミナル共有の強化

### 🟡 共有セッションの暗号化

E2E 暗号化で盗聴防止。

### 🟡 録画・再生 (Session Recording)

- ターミナルセッションを記録 (asciinema 形式互換)
- 再生速度の変更
- 共有リンク生成
- デモ・チュートリアル作成に活用

### 🟡 読み取り専用モード

共有時に閲覧のみ (入力不可) のモードを選択可能に。

### 🟢 インターネット経由の共有

STUN/TURN サーバーで LAN 外のユーザーとも共有。

## 6. AI 連携機能

### 🟡 AI コマンドアシスタント

ターミナル内から自然言語でコマンドを生成。

- `?? ファイルサイズでソートして表示` → `ls -lhS` を提案
- エラー出力の自動解説
- コマンド履歴からのサジェスト

### 🟡 AI チャットサイドバー

ターミナル横にチャットパネルを表示。

- ターミナル出力をコンテキストとして共有
- コマンドの説明・提案
- エラーのトラブルシューティング

### 🟢 スマートオートコンプリート

ターミナル入力に対して AI ベースの補完候補を表示 (GitHub Copilot for CLI 風)。
